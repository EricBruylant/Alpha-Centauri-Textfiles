Read me for SMACX text file typographical update and spelling correction version 1.4

This is a text file update for Alpha Centauri—Alien Crossfire and assumes that you have already installed all the official patches, though it should be compatible with any version. The last changes were made on 4st March 2013.

Although it is primarily focused at correcting spelling errors, it also includes four bug fixes, as well as typographical changes.

The canonical version can be downloaded from the Alpha Centauri 2 forums at the following location:
http://alphacentauri2.info/index.php?topic=2524.15

If you find any errors or have any suggestions for further changes, please post them at the above location.

There is no changelog for this read me. All changes made to game files are indicated here. To determine changes from previous versions, please compare this file with previous update.

In the following changelog The literature refers to:

  • The Alpha Centauri manual;
  • The Alien Crossifre manual;
  • Journey to Alpha Centauri; and
  • Centauri: Arrival

The line numbers are based on the official fully patched files and GooglyBoogly’s update to the SMACX help files. The line number is followed by amendment.

The ditto mark (〃) indicates that the amendement above is repeated.

General changes

Since the original files are written in USA English, I have corrected the spelling on that basis.

  1. Quotation marks—curled
  2. -- changed to —
  3. . . . and ... converted to …
  4. x and * replaced, where appropriate, with the multiplication symbol ×
  5. (c) to ©
  6. Removed any whitespace or carriage returns following # ; This line must remain at end of file
